The goal of the project was to build a dynamic web application using **Node.js**, **Express**, **EJS**, and a **MySQL database**.

1. Clone the repository:

```bash
git clone https://github.com/DianaBosinceanu11/Web-Technologies---Intuitive-Festival.git
cd Web-Technologies---Intuitive-Festival
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
npm start
```

4. Open browser and go to:
```bash
http://localhost:5000
```
